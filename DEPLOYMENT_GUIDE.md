# دليل التشغيل والنشر 📱\n\n## 🔄 التشغيل والإعداد المسبق\n\n### المتطلبات الأساسية\n- **Android Studio Arctic Fox أو أحدث**\n- **Kotlin 1.9.22+** \n- **Gradle 8.3+**\n- **targetSdk 34 (Android 14)**\n- **Java 8+ / OpenJDK 17+**\n- **Git أو أي VCS للإدارة**\n\n### الخطوات التفصيلية للبدء\n\n1. **استنساخ المستودع**:\n   ```bash\n   git clone [repository-url] marina-hotel-app\n   cd marina-hotel-app\n   ```\n\n2. **افتح في Android Studio**:\n   - انتقل إلى: `File → Open → marina-hotel-app`\n   - انتظر حتى يتم تنزيل الـ Gradle sync\n\n3. **التأكد من التبعيات**:\n   - شاهد ملف `build.gradle.kts` للـ App module\n   - تحقق من التبعيات عبر `Build → Make Project`\n\n4. **الإعداد الأولي**:\n   - سيعمل المشروع مباشرةً مع سجلات تسجيل الدخول الافتراضية\n   - المستخدم الافتراضي: `admin` \n   - كلمة المرور الافتراضية: `admin123`\n   - المقترح: إن كنت ستنشره، قم بتعديل كلمات المرور بالضرورة\n\n5. **قواعد البيانات الأولية**:\n   - سيتم إنشاء قاعدة بيانات `hotel_database` تلقائيًا بالسجلات الافتراضية\n   - ستكون هناك 9 غرف افتراضية متوفرة (101-101-109) في ثلاث طوابق\n   - يمكنك التحقق منها في لوحة التحكم فور فتح التطبيق\n\n---\n\n## 📋 ملخص الملامح الرئيسية\n\n### 🎯 المحاور الأساسية\n1. **تسجيل الدخول الآمن** - إدارة المحاسبين والمشرفين\n2. **لوحة التحكم الرئيسية** - نظرة شاملة بآخر ثانية\n3. **الحجوزات والغرف** - الإدارة الكاملة لحجوزات الغرف والنزلاء\n4. **المصاريف والدفعات** - نظام ذكي لحساب الربحية\n5. **التقارير والإحصائيات** - معلومات مفصلة قطرياً\n\n### 🏆 الميزات الفريدة\n- **كل شيء باللغة العربية** - واجهات، رسائل، معالجات\n- **Material 3 الحديث** - تجربة المستخدم مع الروابط الديناميكية\n- **بنية MVVM القوية** - تطوير مبسط وقابل للتوسيع\n- **Room + Flows** - أداء عالي مع التحديثات اللحظية\n- **𝕜 الاختام المؤسسي** - شدات اعطاء الضوابط والترخيص\n\n### 📊 المتغيرات البترولية\n- قاعدة بيانات محلية كاملة (Room)\n- معالجة صفرية للخادوم عن بعد (كامل الخصوصية)\n- تجميع إلكتروني مؤمّن (Kotlin Coroutines)\n- معالجات وحلول للأخطاء شاملة وصيغية\n- حماية من التهجمات مع انتقالات MVC الآمنة\n\n---\n\n## 🚀 التخصيص وال توسيع\n\n### التعديل الموصى به:\n1. **تغيير كلمة مرور المـدير الافتراضية\n   -
