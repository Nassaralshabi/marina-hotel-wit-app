========================================
    كيفية تحويل Marina Hotel إلى EXE
========================================

لديك مجلد phpdesktop-chrome-57.0-rc-php-7.1.3 وهو جاهز للاستخدام.

خطوات التحويل البسيطة:

1. انسخ الملفات التالية إلى مجلد: 
   phpdesktop-chrome-57.0-rc-php-7.1.3\www\

   الملفات المطلوبة:
   ✓ جميع ملفات .php من المجلد الرئيسي
   ✓ مجلد admin (كاملاً)
   ✓ مجلد assets (كاملاً)  
   ✓ مجلد includes (كاملاً)
   ✓ مجلد api (كاملاً)
   ✓ ملف hotel_db.sql
   ✓ ملف setup_exe_database.php

2. أنشئ هذه المجلدات في www\:
   - uploads\
   - uploads\reports\
   - data\

3. تم إعداد settings.json تلقائياً بالإعدادات التالية:
   - العنوان: Marina Hotel Management System
   - الحجم: 1400x900
   - المنفذ: 8080
   - الصفحة الأولى: setup_exe_database.php

4. لتشغيل النظام:
   - اذهب إلى مجلد phpdesktop-chrome-57.0-rc-php-7.1.3
   - شغل phpdesktop-chrome.exe
   - سيفتح النظام تلقائياً

5. معلومات الدخول:
   - اسم المستخدم: admin
   - كلمة المرور: admin123

========================================
    النظام جاهز للاستخدام!
========================================

ملاحظات:
• النظام يعمل بدون إنترنت
• يمكن نسخه إلى أي جهاز كمبيوتر
• يمكن تشغيله من فلاشة USB
• لا يحتاج تثبيت أي برامج إضافية

في حالة المشاكل:
• تأكد من نسخ جميع الملفات
• تأكد من عدم استخدام المنفذ 8080 من برنامج آخر
